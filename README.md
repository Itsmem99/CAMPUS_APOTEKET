Campus Apoteket 🏥
Campus Apoteket är en modern webapplikation designad för att hantera läkemedelsförsörjning och underlätta interaktionen mellan vårdgivare och apotek i en campusmiljö. Appen fokuserar på användarvänlighet, säkerhet och effektiv lagerhantering.

🚀 Funktioner
Läkemedelskatalog: Sök och filtrera bland tillgängliga läkemedel.

Beställningssystem: Enkel process för att lägga och spåra beställningar.

Lagerhantering: Realtidsuppdatering av lagersaldon och varningar vid lågt lager.

Användarroller: Olika behörighetsnivåer för administratörer, personal och kunder.

Responsiv Design: Fullt optimerad för både desktop och mobila enheter.

🛠 Teknisk Stack
Projektet är byggt med följande teknologier:

Frontend: HTML5, CSS3 och JavaScript (ES6+).

Backend: Node.js med Express.

Databas: MySQL / MongoDB (justera beroende på din exakta setup).

Styling: Bootstrap eller skräddarsydd CSS.
